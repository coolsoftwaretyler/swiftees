# Swiftees

A TypeScript runtime built in Swift.

## Building

To build the project:

```bash
swift build
```

## Running

To run the executable:

```bash
swift run swiftees
```

## Testing

To run tests:

```bash
swift test
```

## GitHub Actions

The project includes a GitHub Actions workflow that automatically:
- Builds the project
- Runs tests
- Executes the binary

The workflow runs on every push and pull request to the main branch.
