# Swiftees

A TypeScript runtime built in Swift.

## Prerequisites

- Swift 5.9 or later
- Linux or macOS

## Building

Build the project using Swift Package Manager:

```bash
swift build
```

## Running

Run the executable:

```bash
swift run swiftees
```

## Testing

Run the test suite:

```bash
swift test
```

## CI/CD

This project uses GitHub Actions to automatically build and test on Linux runners. The workflow runs on every push and pull request to the main branch.
